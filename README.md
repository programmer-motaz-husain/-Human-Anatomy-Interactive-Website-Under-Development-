🧬 Human Anatomy Interactive Website (Under Development)
I developed an interactive Biology project where you can click on any organ in a human body diagram 🫀🧠, and it navigates to a dedicated page with detailed information and an associated image of that organ 📄🖼️. This project uses the poly shape technique learned in my Web Full-Stack course, combining HTML and a touch of CSS for styling.

It’s a practical exploration of web interactivity and anatomy visualization, perfect for anyone interested in education technology or interactive learning. 🌐✨

Skills: HTML5, Cascading Style Sheets (CSS)
